# Template v3

Pure HTML, CSS, and JavaScript template for website generation.

## Structure

- `index.html` - Home page
- `contact.html` - Contact page
- `services/service-template.html` - Service page template
- `assets/css/shared.css` - Shared CSS variables and base styles
- `assets/css/styles.css` - Template-specific styles
- `assets/img/` - Image assets directory
- `script.js` - JavaScript functionality
- `prompts/rewrite-site.txt` - LLM prompt template
- `template.tokens.json` - Token mapping configuration

## Features

- Modern, responsive design
- Mobile-first approach
- Smooth scrolling navigation
- Interactive mobile menu
- Form handling
- Dynamic year in footer
- All text content uses prompt format for LLM processing
